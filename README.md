# Titanic Survival Analysis (EDA)

This project explores the Titanic survival dataset and uncovers patterns using visualization and statistical analysis.

---

##  Project Summary:
Includes:
- Data cleaning  
- Exploratory visualizations  
- Correlation analysis  
- Survival insights by gender/class/age  

A great demonstration of EDA and storytelling with data.


---

## Tech Stack :
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- NumPy
